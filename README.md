# 📱 Interactive Prototype Creation

## 🔍 Project Overview
This project focuses on building a clickable interactive prototype to simulate real mobile app interactions. The prototype demonstrates user flow, navigation, and usability using a modern UI/UX approach.

---

## 🎯 Objective
- Convert wireframes into an interactive prototype  
- Simulate real user interactions  
- Improve usability and user experience  

---

## 🧠 Problem Statement
Users often face difficulty understanding app flow from static designs. This project solves that by creating a clickable prototype that visually demonstrates navigation and interactions.

---

## 💡 Solution
Designed an interactive prototype using Figma that allows users to:
- Navigate between screens  
- Interact with UI elements  
- Experience real app-like transitions  

---

## 🧩 Key Features
- Clickable buttons and navigation  
- Smooth transitions (slide, fade)  
- Clear user flow between screens  
- Mobile-friendly design  

---

## 🔄 User Flow
1. Splash Screen →  
2. Login / Signup →  
3. Home Screen →  
4. Feature Screen →  
5. Profile / Settings  

---

## 🎨 Prototype Details
- Tool Used: **Figma**  
- Type: Low to Mid Fidelity Interactive Prototype  
- Screens Included:
  - Splash Screen  
  - Login Screen  
  - Home Screen  
  - Feature Screen  
  - Profile Screen  

---

## 🧪 Usability Testing
Test Scenario:
- User logs in and navigates to main feature  
- User interacts with buttons and transitions  

Feedback:
- Navigation is smooth  
- UI is easy to understand  
- Minor improvements needed in button visibility
